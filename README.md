# WebView Hub

一个轻量级的桌面多网站聚合工具，帮助你在同一界面中并排浏览多个网站，定期追踪行业动态与更新趋势。

---

## 用途

在信息爆炸的时代，追踪某一领域（如 AI 技术、设计资讯、产品动态）往往需要频繁切换多个标签页，容易遗漏更新、打断思路。

**WebView Hub** 将多个同类网站聚合到一个可分栏的工作区中，让你一眼扫览，随时对比：

- **AI 技术趋势**：同时打开 Hacker News、GitHub Trending、V2EX 等，每天一眼看尽更新
- **设计灵感收集**：Dribbble、Behance、Awwwards 并排展示，对比风格趋势
- **市场与竞品监控**：多个竞品官网或媒体报道同屏呈现，追踪动向
- **新闻聚合阅读**：科技、财经、行业媒体集中一处，减少分散注意力

## 快速开始

### 方式一：直接运行（推荐，无需安装任何环境）

1. 下载并解压 `dist/WebView-Hub-win.zip`
2. 双击解压后目录下 `WebView Hub.exe` 即可运行

> 无需安装 Node.js、Python 或任何其他软件。

### 方式二：从源码启动（开发用）

**环境要求**：Node.js 18+

```bash
# 首次运行先安装依赖
npm install

# 启动
npm start
```

或直接双击 `start_electron.bat`（会自动安装依赖并启动）。

## 界面说明

```
┌─────────────────────────────────────────────────────┐
│  [AI 资讯] [工作] [+]                    ↓导入  ↑导出 │  ← 标签栏
├──────────────────────────┬──────────────────────────┤
│                          │   侧边格口 1              │
│                          ├──────────────────────────┤
│       主格口              │   侧边格口 2              │
│                          ├──────────────────────────┤
│                          │   侧边格口 3              │
├──────────────────────────┴──────────────────────────┤
│                    + 添加侧边网站                     │
└─────────────────────────────────────────────────────┘
```

- **左侧主格口**：主要浏览区域，宽度占比较大
- **右侧侧边格口**：可添加多个，每个格口独立显示一个网站
- **分割线**：左右分割线可左右拖拽，上下分割线可上下拖拽

## 功能说明

### 标签分组

| 操作 | 说明 |
|------|------|
| 点击 `+` 按钮 | 新建分组 |
| 双击标签名 | 重命名分组 |
| 点击标签 `×` | 删除分组（至少保留一个） |

每个分组独立保存一套主格口 + 侧边格口配置，切换标签即切换整套布局。

### 添加 / 编辑网站

- 悬停到格口上，右上角会出现 **✏ 编辑** 和 **✕ 删除** 按钮
- 点击底部 **＋ 添加侧边网站** 可在右侧新增格口
- 点击主格口空白处的 **＋ 添加网站** 可设置主格口网站
- 输入网址时无需加 `https://`，会自动补全

### 布局调整

- **拖拽垂直分割线**（左右两栏之间）：调整主格口与侧边栏的宽度比例
- **拖拽水平分割线**（侧边格口之间）：调整各格口的高度比例
- **双击侧边格口标题**（悬停后显示）：将该格口与主格口互换位置，快速聚焦

### 配置迁移

| 按钮 | 说明 |
|------|------|
| `↓ 导入` | 从本地 JSON 文件导入配置，覆盖当前所有分组和网站 |
| `↑ 导出` | 将当前配置导出为 JSON 文件保存到本地 |

配置同时自动保存在程序目录下的 `setting/config.json`，可直接复制该文件进行备份或在多台机器间同步。

---

## 注意事项

- 部分网站（如百度、哔哩哔哩等）设有反嵌入保护，可能无法正常显示，属正常现象
- 配置文件保存在 `setting/config.json`，删除该文件会重置为默认配置
- 首次启动时会自动加载默认示例网站（Hacker News、Lobste.rs 等），可自行替换

---

## 功能特性汇总

- 分栏布局，所有分割线均可拖拽调整
- 每个格口为完整嵌入页面，可直接交互
- 多标签分组，一键切换不同主题布局
- 双击格口标题可与主区域快速互换
- 支持导入/导出 JSON 配置，跨设备同步
- 配置自动持久化，重启后恢复上次布局
